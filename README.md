# Java 注册登录窗口系统

这是一个使用Java Swing库设计的图形化用户界面（GUI）注册登录系统。

## 功能特性

### 1. **登录界面** (LoginFrame.java)
- 用户名和密码输入框
- 登录和注册按钮
- 登录状态提示消息
- 输入验证功能

### 2. **注册界面** (RegisterFrame.java)
- 用户名、邮箱、密码输入框
- 密码确认功能
- 完整的输入验证：
  - 用户名长度至少3个字符
  - 密码长度至少6个字符
  - 邮箱格式验证
  - 密码一致性验证
  - 用户名唯一性检查

### 3. **用户管理** (UserManager.java)
- 用户注册管理
- 用户凭证验证
- 用户数据存储（基于内存的HashMap）

### 4. **主程序** (Main.java)
- 应用启动入口
- 初始化用户管理和UI

## 编译运行

## 技术栈

- **Java Swing**: GUI组件库
- **GridBagLayout**: 灵活的布局管理器
- **HashMap**: 用户数据存储
- **正则表达式**: 邮箱格式验证

## 使用流程

1. 启动应用后，显示登录界面
2. 可以使用测试账户登录，或点击"注册新用户"创建新账户
3. 注册时需要填写所有必需信息
4. 注册成功后返回登录界面进行登录
5. 登录成功会显示欢迎提示

## 功能说明

### 登录功能
- 验证用户名和密码
- 登录成功显示欢迎信息
- 登录失败显示错误提示

### 注册功能
- 用户名验证（3个字符以上，不能重复）
- 邮箱格式验证
- 密码强度检查（6个字符以上）
- 密码一致性验证
- 实时反馈验证结果
