## 🚀 部署信息

### 系统要求
- Java 8 或更高版本
- 1024 MB RAM (建议)
- 500 MB 磁盘空间 (建议)
- 任何现代浏览器

### 启动步骤
1. 确保 Java 已安装: `java -version`
2. 进入项目目录
3. 运行: `./start-webapi.sh`
4. 打开浏览器访问: `http://localhost:8080`

### 默认配置
- **服务器端口：** 8080
- **测试账户：** admin / admin123
- **数据存储：** 内存 (HashMap)
